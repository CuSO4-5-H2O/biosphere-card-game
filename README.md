# 生息界：非洲草原 Demo 1.2

一个以生态系统自动演替为核心的回合制卡牌原型。玩家用有限部署能量建立生产者、草食动物与捕食者，并让系统通过 10 回合无人干预稳定性考核。

## 在线体验

<https://cuso4-5-h2o.github.io/biosphere-card-game/>

## 本地运行

```powershell
node server.mjs
```

浏览器打开 `http://127.0.0.1:4173`。

## Demo 1.2 内容

- 8 回合构筑阶段，加 10 回合无人干预稳定性考核
- 开局规则、胜利条件与基础策略提示
- 适配常见桌面视口的一屏游戏布局
- 河岸草甸、开阔草原、金合欢灌丛三个区域
- 草、金合欢、瞪羚、斑马、蓝角马、狮子、斑鬣狗七种生物
- 10 种带独特遗传特性的生物卡牌变体
- 季风雨带、黑土层、白蚁网络三张开局环境牌
- 降雨、干旱、迁徙、草原火、疫病、养分输入等随机事件
- 卡牌抽取、选择、投放和生态结算动效
- 营养级历史曲线与结算物种数量统计图

## 操作

1. 阅读规则与胜利条件。
2. 从三张全局环境牌中选择一张。
3. 点击生物牌，再点击地图中高亮的适生区域。
4. 根据能量、区域承载力和食物链结构决定是否继续部署。
5. 点击“结束部署”，观察本回合自动结算。
6. 第 8 回合后启动 10 回合无人干预考核。

通关要求是在考核期间保持生产者、草食动物和捕食者不断层，后五回合进入合理波动范围，并控制有机残留。
