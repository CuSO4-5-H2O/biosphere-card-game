# 生息界：非洲草原 Demo 1.1

一个以生态系统自动演替为核心的回合制卡牌原型。玩家用有限部署能量建立生产者、草食动物与捕食者，并让系统通过 10 回合无人干预稳定性考核。

## 运行

```powershell
node server.mjs
```

浏览器打开 `http://127.0.0.1:4173`。

## Demo 1.1 内容

- 8 回合构筑阶段，加 10 回合无人干预稳定性考核
- 河岸草甸、开阔草原、金合欢灌丛三个区域
- 草、金合欢、瞪羚、斑马、蓝角马、狮子、斑鬣狗七种生物
- 10 种带独特遗传特性的生物卡牌变体
- 季风雨带、黑土层、白蚁网络三张开局环境牌
- 降雨、干旱、迁徙、草原火、疫病、养分输入等随机事件
- 植物生长、自然迁徙、采食、捕食、繁殖、死亡与分解闭环
- 营养级监控、生态预估、事件日志和 18 回合历史曲线

## 操作

1. 开局从三张全局环境牌中选择一张。
2. 点击下方生物牌，再点击地图中高亮的适生区域。
3. 根据能量、区域承载力和食物链结构决定是否继续部署。
4. 点击“结束部署”，观察本回合自动结算。
5. 第 8 回合后启动 10 回合无人干预考核。

通关要求是在考核期间保持生产者、草食动物和捕食者不断层，后五回合进入合理波动范围，并控制有机残留。
